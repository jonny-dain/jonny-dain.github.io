Campus Commandos

Campus Commandos is a side-scrolling 2D high-action shooter set at the University of Birmingham, where players stop an alien attack using various weapon types across three game modes. The game features a destructible environment with blocks that can be shot, blown up, and manipulated to create tactical positions. It follows an NES pixel art style and offers both solo and multiplayer options, including cooperative, competitive, and survival modes.

  • Campaign Mode: Follow the main story through various Birmingham locations and fantastical places. Play solo or with up to 4 players online.
  • Competitive Mode: Fight in an online arena against 2-4 players to gain points by defeating each other.
  • Survival Mode: Single-player mode with procedurally generated levels. Health and ammo carry over, challenging players to survive as many levels as possible.

For more information, check out the GitHub repository

Gradrotate

Gradrotate is an online application designed to automate the allocation of interns to suitable placement positions, enhancing productivity for both interns and employers. It utilises algorithms like Gale-Shapely, Hungarian, Pareto, and Random Serial Dictatorship to ensure accurate matches, reducing bias and increasing efficiency compared to manual processes.

This application was created for Vodafone Group's 2023 Graduate intake to streamline their assignment to the most appropriate rotational positions and has now been integrated into the Vodafone workflow for all graduates during onboarding.

For more information, check out the GitHub repository

Languages
Python, HTML, CSS, JavaScript
Tags
Platform using sorting algorithms to place interns in positions.
Commonwealth Games booking platform

Challenged by the University, This project tackles the issue of event attendance during the COVID-19 pandemic, particularly for the 2022 Commonwealth Games in Birmingham. It proposes a flexible seat allocation strategy to maximise attendance while adhering to government guidelines to minimise viral spread. The solution encompasses a booking and allocation subsystem, ensuring a seamless user journey from ticket booking to seating arrangements.

PetSense

PetSense is an IoT pet tracking system that simplifies monitoring pets' welfare. Featuring GPS-equipped collar, automated feeder, and pet door APIs, it tracks activity, health, and safety while easing owners' routines. The collar monitors location and health, geofencing alerts if pets stray, while the feeder ensures proper nutrition and the door allows controlled access. All elements are managed via a mobile app for real-time updates and control.

Languages
Dart, Flutter
Tags
IOT application for tracking pet habits
